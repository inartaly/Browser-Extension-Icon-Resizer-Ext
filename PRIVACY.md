# Privacy Policy for Browser Extension Icon Resizer

## Data Collection
This extension does not collect, store, or transmit any personal data.  
No analytics, tracking scripts, or user identifiers are used.

## How the Extension Works
The extension loads a remote web application hosted on GitHub Pages at:

https://inartaly.github.io/Browser-Extension-Icon-Resizer/

All functionality—including image processing and icon generation—is performed entirely within the user's browser. Images are never uploaded to a server or transmitted over the network.

## Permissions
This extension does not request any Chrome permissions, host permissions, or access to user data.  
It only opens the remote interface where the resizing tool is hosted.

## Remote Code
Because the resizing logic is hosted on GitHub Pages, the extension loads and executes remote code from that domain. No third‑party domains are used.

## Data Handling
All image processing happens locally in the browser session.  
No files, metadata, or user information leave the device.

## Contact
For questions or support, please visit the project repository:

https://github.com/inartaly/Browser-Extension-Icon-Resizer-Ext
